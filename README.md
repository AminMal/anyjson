# AnyJson

AnyJson provides a unified syntax for all the different JSON libraries in Scala.
Well, not all of them, the ones who follow the type-class formatting pattern.

## Supported libraries
AnyJson currently supports:

- [X] PlayJson
- [X] Circe
- [X] Json4s
- [X] Json4sJackson
- [X] Jsoniter
- [X] SprayJson
- [X] ZIO Json
